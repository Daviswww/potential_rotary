# Potential Rotary

## Using Django
### Creating a project
```
$ django-admin startproject mysite
```


### Runserver
```bash
$ python3 manage.py runserver
```

### Migrate
```bash
$ python3 manage.py migrate
```

## Creating Virtual Environments
### Install Virtualenv
```bash
$ pip install virtualenv
```

### Create env
```bash
$ virtualenv env
```

### On Unix or MacOS, run:
```bash
$ source env/bin/activate
```
